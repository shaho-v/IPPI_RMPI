# Statistics Canada Industrial Products Price Index and Raw Material Price Index

This R Markdown file pulls data for these price indexes from Statistics Canada, calculates month-over-month and year-over-year changes, and then presents the top monthly and yearly changes (along with the greatest decreases in IPPI products) in a table.
